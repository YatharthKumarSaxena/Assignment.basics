Q6:- How many of these can be avariable name?
     1.) 01Pwskills
     2.) _FLOAT
     3.) int
     4.) FLOAT
     5.) You will succeed
Ans:- Only two variable names are correct that is FLOAT and _FLOAT
      01Pwskills is incorrect because variable name is starting with number that is not allowed
      int is aspecial keyword hence this variable name is also invalid
      You will succeed is also incorrect due to the spaces present in the variable name
